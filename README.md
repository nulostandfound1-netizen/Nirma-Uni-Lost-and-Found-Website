# Nirma-Uni-Lost-and-Found-Website
Nirma University Lost and Found dashboard is a student made website to streamline the lost and found process. The following are the major features of this website: -

1. Secure login with Official Nirma Uni Email IDs
2. Easy registration for lost and found items on the website
3. Google gemini integration for efficient searching
4. Firebase realtime database integration for saving and displaying the data
5. Email JS integration to send Emails to the users on various scenarios
6. Added the case of dispute if an item is been disputed for ownership between 2 or more people
